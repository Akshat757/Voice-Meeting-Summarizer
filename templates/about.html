About the Project

AVA, a voice assistant, is envisioned to be a multifunctional tool aimed at simplifying note-taking and summarization tasks. Leveraging Python for backend logic and integrating various libraries such as TensorFlow for AI capabilities and Flask for web development, the project's goal is to enhance productivity by providing a convenient tool for note-taking and summarization. It aims to facilitate effective communication and collaboration by streamlining tasks such as:

  • Recording audio
  • Converting audio into text
  • Storing text along with summaries in a database
  • Providing CRUD operations for managing records
  • Offering real-time updates during recording
  • Extracting keywords from recordings
  • Enabling email sharing functionality

AVA empowers users to manage meeting notes and communication effectively without worrying about tedious note-taking processes.
